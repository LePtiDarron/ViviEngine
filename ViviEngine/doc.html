Core Classes

Vivi Engine provides three main classes that form the backbone of any game:

Game

The main engine class that manages the game loop, window, and scene transitions.

Constructor

Game(width=800, height=600, title="ViviEngine Game", fps=60)

Key Methods

add_scene(name, scene) - Register a scene with a name
switch_scene(name) - Schedule scene change for end of frame
init_scene(name) - Immediately initialize and activate scene
run() - Start the main game loop
get_delta_time() - Get frame time in seconds
Scene

Represents a game screen or level. Manages entities and handles game logic for a specific state.

Override Methods

create() - Called when scene is initialized
step() - Called every frame for logic updates
draw() - Called every frame for rendering
cleanup() - Called when scene is destroyed

Entity Management

add_entity(entity) - Add entity to scene
remove_entity(entity) - Mark entity for removal
get_entities_of_type(type) - Find all entities of specific type
count_entities_of_type(type) - Count entities of specific type

Properties

background_color - RGB tuple for background color
Entity

Base class for all game objects. Provides position, sprites, collision detection, and lifecycle management.

Constructor

Entity(x=0, y=0)

Position & Transform

x, y - Current position
xprevious, yprevious - Previous frame position
image_xscale, image_yscale - Sprite scaling (1.0 = normal)
image_angle - Rotation in degrees
image_alpha - Transparency (0.0 to 1.0)

Rendering

sprite_index - Name of sprite to display
depth - Render depth (higher = in front)
visible - Whether entity is drawn

State

active - Whether step() is called
scene - Reference to parent scene
id - Unique identifier

Key Methods

create() - Override for initialization
step() - Override for per-frame logic
draw() - Override for custom rendering
destroy() - Mark for destruction
set_sprite(name) - Change sprite

Collision Detection

bbox_collision(other) - Check collision with another entity
point_in_bbox(x, y) - Check if point is inside entity
distance_to(other) - Calculate distance to another entity
direction_to(other) - Calculate angle to another entity

Utility Functions

Vivi Engine provides a comprehensive set of utility functions for common game development tasks:

Asset Management
load_assets(folder="assets") - Load all sprites, sounds, and fonts from folder
get_sprite(name) - Retrieve loaded sprite by name
get_sound(name) - Retrieve loaded sound by name
get_font(name) - Retrieve loaded font by name
Drawing Functions
draw_clear(color) - Clear screen with RGB color
draw_sprite(x, y, name, xscale=1, yscale=1, angle=0) - Draw sprite with transformations
draw_text(x, y, text, scale=1, font=None) - Draw text string
draw_rectangle(x, y, width, height, filled=True) - Draw rectangle
draw_circle(x, y, radius, filled=True) - Draw circle
draw_line(x1, y1, x2, y2, width=1) - Draw line
draw_set_color(rgb_tuple) - Set drawing color
draw_set_alpha(alpha) - Set drawing transparency (0.0-1.0)
Input Handling

Keyboard

keyboard_check_pressed(key) - True if key was just pressed this frame
keyboard_check(key) - True if key is currently held down
keyboard_check_released(key) - True if key was just released this frame

Mouse

mouse_check_pressed(button) - True if mouse button just pressed (1=left, 2=middle, 3=right)
mouse_check(button) - True if mouse button held down
mouse_check_released(button) - True if mouse button just released
mouse_get_x(), mouse_get_y() - Get current mouse position
Audio
play_sound(name, volume=1.0) - Play sound effect with volume control
stop_sound(name) - Stop playing sound
Math & Utilities
random_range(min, max) - Generate random float between min and max
random_int(min, max) - Generate random integer between min and max (inclusive)
clamp(value, min, max) - Constrain value within range
lerp(start, end, factor) - Linear interpolation between two values
distance(x1, y1, x2, y2) - Calculate Euclidean distance between points
point_direction(x1, y1, x2, y2) - Calculate angle between points in degrees
lengthdir_x(length, direction) - Get X component of vector
lengthdir_y(length, direction) - Get Y component of vector
Surface Management
surface_create(width, height) - Create new drawing surface
surface_set_target(surface) - Set drawing target to surface
surface_reset_target() - Reset drawing target to screen
surface_draw(x, y, surface, xscale=1, yscale=1) - Draw surface to current target
surface_destroy(surface) - Free surface memory
Camera System
camera_create() - Create new camera, returns camera ID
camera_set_view_size(id, width, height) - Set camera view dimensions
camera_set_view_pos(id, x, y) - Set camera position in world
camera_set_view_port(id, x, y, width, height) - Set camera viewport on screen
camera_set_active(id) - Make camera active
camera_get_view_pos(id=None) - Get camera position
Entity Management
entity_create(x, y, entity_class) - Create and add entity to current scene
entity_destroy(entity) - Mark entity for destruction
Window Management
window_get_size() - Returns tuple (width, height) of window
window_set_size(width, height) - Resize game window

Tutorial 1: Your First Game

Learn the basics by creating a simple interactive scene with player movement.

Step 1: Setup Project

Create a new folder for your game and add the main.py file:

from ViviEngine import * import pygame import os # Load assets assets_path = os.path.join(os.path.dirname(__file__), "assets") load_assets(assets_path)

Step 2: Create Player Entity

class Player(Entity): def create(self): super().create() # Set sprite if you have one, otherwise we'll draw a rectangle self.speed = 200 self.color = (255, 100, 100) # Red color def step(self): super().step() dt = self.scene.game.get_delta_time() # Movement with arrow keys or WASD if keyboard_check(pygame.K_LEFT) or keyboard_check(ord('a')): self.x -= self.speed * dt if keyboard_check(pygame.K_RIGHT) or keyboard_check(ord('d')): self.x += self.speed * dt if keyboard_check(pygame.K_UP) or keyboard_check(ord('w')): self.y -= self.speed * dt if keyboard_check(pygame.K_DOWN) or keyboard_check(ord('s')): self.y += self.speed * dt # Keep player on screen self.x = clamp(self.x, 25, window_get_size()[0] - 25) self.y = clamp(self.y, 25, window_get_size()[1] - 25) def draw(self): super().draw() # Draw a simple rectangle if no sprite if not self.sprite_index: draw_set_color(self.color) draw_rectangle(self.x - 25, self.y - 25, 50, 50, True)

Step 3: Create Game Scene

class GameScene(Scene): def create(self): super().create() self.background_color = (50, 150, 200) # Sky blue # Create player in center of screen player = Player(400, 300) self.add_entity(player) def step(self): super().step() # Exit game with Escape if keyboard_check_pressed(pygame.K_ESCAPE): self.game.quit() def draw(self): super().draw() # Draw UI text draw_set_color((255, 255, 255)) draw_text(10, 10, "Use WASD or Arrow Keys to move", scale=1.5) draw_text(10, 40, "Press ESC to exit", scale=1)

Step 4: Run the Game

# Create game instance game = Game(800, 600, "My First Vivi Game", 60) # Create and add scene game_scene = GameScene() game.add_scene("game", game_scene) # Start the game game.init_scene("game") game.run()

Congratulations! You now have a working game with player movement. Try adding sprites by placing PNG files in an "assets/images/" folder!

Tutorial 2: Adding Enemies and Collision

Expand your game with enemy entities and collision detection.

Step 1: Create Enemy Entity

class Enemy(Entity): def create(self): super().create() self.speed = 100 self.color = (255, 50, 50) # Red self.target = None def step(self): super().step() dt = self.scene.game.get_delta_time() # Find player target if not self.target: players = self.scene.get_entities_of_type(Player) if players: self.target = players[0] # Move toward player if self.target: direction = self.direction_to(self.target) vx = lengthdir_x(self.speed, direction) vy = lengthdir_y(self.speed, direction) self.x += vx * dt self.y += vy * dt def draw(self): super().draw() if not self.sprite_index: draw_set_color(self.color) draw_circle(self.x, self.y, 20, True)

Step 2: Add Collision Detection

# Update Player class step method def step(self): super().step() dt = self.scene.game.get_delta_time() # ... existing movement code ... # Check collision with enemies enemies = self.scene.get_entities_of_type(Enemy) for enemy in enemies: if self.bbox_collision(enemy): # Game over - restart scene self.scene.game.switch_scene("game") break

Step 1: Create Bullet Entity

class Bullet(Entity): def create(self): super().create() self.speed = 400 self.direction = 0 # Will be set when created self.color = (255, 255, 100) # Yellow def step(self): super().step() dt = self.scene.game.get_delta_time() # Move in direction vx = lengthdir_x(self.speed, self.direction) vy = lengthdir_y(self.speed, self.direction) self.x += vx * dt self.y += vy * dt # Remove if off screen w, h = window_get_size() if (self.x < -50 or self.x > w + 50 or self.y < -50 or self.y > h + 50): self.destroy() # Check collision with enemies enemies = self.scene.get_entities_of_type(Enemy) for enemy in enemies: if self.distance_to(enemy) < 30: # Simple distance check enemy.destroy() self.destroy() # play_sound("explosion") # Uncomment if you have sound break def draw(self): super().draw() if not self.sprite_index: draw_set_color(self.color) draw_circle(self.x, self.y, 5, True)

Step 2: Add Shooting to Player

# Update Player class class Player(Entity): def create(self): super().create() self.speed = 200 self.color = (255, 100, 100) self.shoot_cooldown = 0 self.shoot_delay = 0.2 # Can shoot every 0.2 seconds def step(self): super().step() dt = self.scene.game.get_delta_time() # ... existing movement code ... # Update shoot cooldown if self.shoot_cooldown > 0: self.shoot_cooldown -= dt # Shooting with mouse if mouse_check(1) and self.shoot_cooldown <= 0: # Left mouse button mouse_x = mouse_get_x() mouse_y = mouse_get_y() # Calculate direction to mouse direction = point_direction(self.x, self.y, mouse_x, mouse_y) # Create bullet bullet = Bullet(self.x, self.y) bullet.direction = direction self.scene.add_entity(bullet) # Reset cooldown self.shoot_cooldown = self.shoot_delay # Play sound effect # play_sound("shoot") # Uncomment if you have sound

class MenuScene(Scene): def create(self): super().create() self.background_color = (20, 30, 50) # Dark blue self.title_flash = 0 def step(self): super().step() dt = self.game.get_delta_time() self.title_flash += dt * 3 # Flash animation # Start game with SPACE if keyboard_check_pressed(pygame.K_SPACE): self.game.switch_scene("game") # Exit with ESC if keyboard_check_pressed(pygame.K_ESCAPE): self.game.quit() def draw(self): super().draw() w, h = window_get_size() # Title with flash effect flash = abs(math.sin(self.title_flash)) draw_set_alpha(0.7 + flash * 0.3) draw_set_color((255, 200, 100)) draw_text(w//2 - 150, h//2 - 100, "SPACE SHOOTER", scale=3) # Instructions draw_set_alpha(1.0) draw_set_color((255, 255, 255)) draw_text(w//2 - 100, h//2, "Press SPACE to Start", scale=1.5) draw_text(w//2 - 80, h//2 + 40, "Press ESC to Exit", scale=1)

Step 2: Create Game Over Scene

class GameOverScene(Scene): def create(self): super().create() self.background_color = (50, 20, 20) # Dark red self.final_score = 0 # Will be set by game scene def step(self): super().step() # Restart with SPACE if keyboard_check_pressed(pygame.K_SPACE): self.game.switch_scene("game") # Back to menu with ESC if keyboard_check_pressed(pygame.K_ESCAPE): self.game.switch_scene("menu") def draw(self): super().draw() w, h = window_get_size() # Game Over text draw_set_color((255, 100, 100)) draw_text(w//2 - 120, h//2 - 100, "GAME OVER", scale=3) # Score draw_set_color((255, 255, 255)) draw_text(w//2 - 80, h//2 - 20, f"Score: {self.final_score}", scale=2) # Instructions draw_text(w//2 - 100, h//2 + 40, "SPACE: Play Again", scale=1) draw_text(w//2 - 80, h//2 + 70, "ESC: Main Menu", scale=1)

Camera Following

# In your GameScene create method def create(self): super().create() # ... existing code ... # Create camera self.camera = camera_create() camera_set_view_size(self.camera, 800, 600) camera_set_view_port(self.camera, 0, 0, 800, 600) camera_set_active(self.camera) def step(self): super().step() # ... existing code ... # Follow player with camera players = self.get_entities_of_type(Player) if players: player = players[0] # Smooth camera following cam_x, cam_y = camera_get_view_pos(self.camera) target_x = player.x - 400 # Center on player target_y = player.y - 300 new_x = lerp(cam_x, target_x, 0.1) # Smooth interpolation new_y = lerp(cam_y, target_y, 0.1) camera_set_view_pos(self.camera, new_x, new_y)

Particle System

class Particle(Entity): def create(self): super().create() self.vx = random_range(-100, 100) self.vy = random_range(-100, 100) self.life = random_range(1.0, 2.0) self.max_life = self.life self.size = random_range(3, 8) self.color = (255, random_int(100, 255), 0) # Orange-ish def step(self): super().step() dt = self.scene.game.get_delta_time() # Move self.x += self.vx * dt self.y += self.vy * dt # Fade over time self.life -= dt self.image_alpha = self.life / self.max_life if self.life <= 0: self.destroy() def draw(self): super().draw() alpha = self.image_alpha draw_set_alpha(alpha) draw_set_color(self.color) draw_circle(self.x, self.y, self.size * alpha, True) draw_set_alpha(1.0) # Reset alpha # Create explosion particles def create_explosion(scene, x, y, count=10): for i in range(count): particle = Particle(x, y) scene.add_entity(particle)
